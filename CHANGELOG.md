## [1.0.0] - 2024-10-23

### Added
- Initial release of Debian post-installation script
- Support for Sway, GNOME, and KDE desktop environments
- Automatic hardware detection and optimizations (GPU, SSD, power management)
- Comprehensive package installation system with apt and Flatpak support
- Modular configuration management for desktop/notebook setups
- Custom themes, icons, and fonts installation
- Browser customization scripts for Firefox-based browsers
- Arkenfox user.js integration for Firefox ESR
- Systemd services for automatic theme and wallpaper switching
- Interactive whiptail-based user interface
- Comprehensive logging and backup system
- Support for special packages (VeraCrypt, AppImageLauncher, Helium)
